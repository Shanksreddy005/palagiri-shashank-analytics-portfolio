# Analytics Portfolio Website

A premium, single-page, recruiter-focused portfolio website for **Palagiri Shashank Reddy**, optimized for Data Analyst, Business Analyst, Product Analyst, and Founder's Office (Analytics) opportunities.

## Technologies Used
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Interactions**: Framer Motion
- **Icons**: Lucide React
- **Theme**: Apple-inspired minimalist aesthetic

## Key Design & Recruiting Features
1. **Single-Page Scrolling Experience**: Floating navigation tracking and smooth scrolling.
2. **Featured Analytics Case Studies**: Detailed breakdowns for **Food Delivery Platform Analytics** and **Financial Market Trend Dashboard**.
3. **Tableau Public Embed Integration**: Interactive dashboard cards with direct "Open in Tableau Public" links.
4. **Master Resume Alignment**: Skill catalogs, certifications, and experience timelines matching master resume specifications.
5. **Impact KPIs**: Visual counters representing processed records, SQL query count, and order volumes.

## Development Setup

### Installation
From the root workspace directory, run:
```bash
npm install --legacy-peer-deps
```

### Run Locally
Start the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to inspect the application.

## Vercel Deployment Instructions
To deploy to production on Vercel:
1. Initialize a Git repository and commit the files.
2. Push to a GitHub repository named `palagiri-shashank-analytics-portfolio`.
3. Go to [Vercel](https://vercel.com) and click **Add New Project**.
4. Select your `analytics-portfolio` repository.
5. Keep default settings (Next.js is auto-detected) and click **Deploy**.
